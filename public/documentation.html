Movie_API Documentation

Movie_API is the serverside component of a web application named myFlix. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Data Formats for HTTP Requests/Responses
Description Method URL Request Parameter Body of Request Body of Response
Return a list of ALL movies to the user GET /movies None None A JSON object holding data about all movies
Return data (description, genre, director, image URL, whether it’s featured or not) about a single movie by title to the user GET /movies/[Title] /movies/:Title None A JSON object holding data about a single movie. Example:
{ "Description": "Two imprisoned men bond over a number of years, finding solace and eventual redemption through acts of common decency.",
"Genre": " Drama",
"Director": "Frank Darabont",
"image-URL": "https://m.media-amazon.com/images/M/MV5BNjQ2NDA3MDcxMF5BMl5BanBnXkFtZTgwMjE5NTU0NzE@._V1_CR0,60,640,360_AL_UX477_CR0,0,477,268_AL_.jpg"
}
Return data about a genre (description) by name/title (e.g., “Thriller”) GET /movies/genres/[Genre] /movies/genres/:Genre None A JSON object holding data about a single genre
Return data about a director (bio, birth year, death year) by name GET /movies/director/[Name] /movies/director/:Name None A JSON object holding data about the director
Allow new users to register POST /users None A JSON object holding data about the user to create. A JSON object holding data about the user that was added.
Allow users to update their user info (username) PUT /users/[Username] /users/:Username A JSON object holding (required) data about the user to update. A JSON object holding data that was updated.
Allow users to add a movie to their list of favorites (showing only a text that a movie has been added—more on this later) POST /users/[Username]/favorites/[MovieID] /users/:Username/favorites/:MovieID None A JSON object holding data about the user with movies on favorite list.
Allow users to remove a movie from their list of favorites (showing only a text that a movie has been removed—more on this later) DELETE /users/[Username]/favorites/[MovieID] /users/:Username/favorites/:MovieID None A JSON object holding data about the updated favorite list.
Allow existing users to deregister (showing only a text that a user email has been removed—more on this later) DELETE /users/[Username] /users/:Username None A text message indicating that the user was deleted
